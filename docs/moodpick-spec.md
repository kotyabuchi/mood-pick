# MoodPick - プロダクト仕様書

## 概要

気分ベースで「今日何見る？」を決めるウォッチリスト管理アプリ。
ストリーミングサービスを開く前に、自分のリストから作品を選ぶ体験を提供。

### 解決する課題

- ブラウザのタブに「見たい作品」を溜め込む → 自動クローズで消える
- ブックマークは墓場化する
- Netflix等を開くと、プラットフォームのおすすめに流されて自分の見たかった作品を忘れる
- 配信終了に気づかず見逃す

### コアコンセプト

**「ストリーミングを開く前に、自分のリストから選ぶ」**

---

## ターゲット市場

### 日本市場データ

- 動画配信サービス利用者: 3,450万人（2024年）→ 4,120万人（2027年予測）
- 71.5%がスマートフォンで視聴
- Filmarks: 9.2億月間ユーザー、1.6億月間PV

### 競合分析

| サービス | 強み | MoodPickとの差別化 |
|---------|------|-------------------|
| Filmarks | レビュー、大規模DB | 気分ベース選択なし |
| Annict | アニメ特化、エピソード管理 | 気分ベース選択なし |
| Letterboxd | 映画コミュニティ | 配信終了アラートなし |

**既存サービスに無い機能:**
- 気分・時間・集中度での絞り込み
- 配信終了「事前」アラート
- ストリーミングを開く前に決める体験

## 機能要件

### Phase 1 MVP

#### 登録系
- [ ] タイトル入力で作品検索（TMDb API）
- [ ] 検索結果から「見たい」に追加
- [ ] 気分タグ自動判定（ジャンル→気分マッピング）
- [ ] 気分タグ手動調整

#### 選択系
- [ ] 気分選択（5種類）
  - 😆 興奮したい
  - 😢 切ない気分
  - 😂 笑いたい
  - 🤔 考えたい
  - 🫠 まったりしたい
- [ ] 時間選択（4種類）
  - 30分
  - 1時間
  - 2時間
  - 無制限（∞）
- [ ] 集中度選択（2種類）
  - がっつり集中
  - ながら見OK
- [ ] 条件に合う作品をリスト表示
- [ ] 作品選択 →「今すぐ見る」

#### 記録系
- [ ] ステータス管理
  - 見たい → 視聴中 → 見た
  - 視聴中 → 合わなくてやめた（離脱記録）
  - 視聴中 → 見たいに戻す
  - 削除
- [ ] 視聴中: 自由メモ欄（任意、例:「15話まで見た」「シーズン2の途中」）
  - ※ 話数の厳密な管理はしない。進捗はストリーミング側が管理する前提
- [ ] 見た: 評価（星5段階、任意）
- [ ] 見た: 感想メモ（任意）

#### 管理系
- [ ] 見たいリスト一覧
- [ ] 視聴中リスト一覧
- [ ] 見たリスト一覧
- [ ] 並び替え（追加日/タイトル/終了日/視聴日）
- [ ] 絞り込み（映画/ドラマ/アニメ）
- [ ] 作品削除

#### 通知系
- [ ] 配信終了アラート（7日前、3日前、前日）
- [ ] 通知一覧画面

#### ソーシャル系
- [ ] ユーザー登録/ログイン
- [ ] フォロー/フォロワー
- [ ] 招待リンク生成
- [ ] 招待リンクからの登録 → フォロー確認
- [ ] リスト公開スコープ設定（非公開/フォロワーのみ/全体公開）
- [ ] フィード（友達のアクティビティ）
- [ ] 友達が見た作品・評価を見る
- [ ] 友達におすすめする

## 技術スタック

### フロントエンド
- React + TypeScript
- Tailwind CSS
- PWA（モバイル + デスクトップ対応）

### バックエンド
- Supabase
  - Authentication（ユーザー認証）
  - Database（PostgreSQL）
  - Hosting

### 外部API

| API | 用途 | 料金 |
|-----|------|------|
| TMDb API | 作品検索、メタデータ取得 | 無料 |
| Streaming Availability API | 配信状況、終了予定日取得 | 無料100回/日 |

### Supabase制約（無料プラン）
- 2プロジェクトまで
- 500MB DB
- 1GB ストレージ
- 7日間非アクティブで自動停止

### ナビゲーション構造

| タブ | アイコン | 内容 |
|------|---------|------|
| ホーム | 🏠 | 視聴中 + 配信終了アラート |
| リスト | 📚 | 見たい/視聴中/見た |
| 検索（中央） | 🔍 | 大きめ丸ボタン、タップでボトムシート表示 |
| フィード | 👥 | 友達のアクティビティ |
| マイページ | 👤 | 設定、フォロー管理 |

※ 検索ボタンは中央に大きめの丸型（#FF6B00）で配置し、ナビバーから少し飛び出す
※ 検索ボトムシートには「タイトル検索」と「気分で探す」の2つの導線を統合

### 画面一覧

1. **ホーム画面**
   - 視聴中セクション（横スクロール）
   - 配信終了まもなくセクション（横スクロール）
   - ※ 気分選択UIは検索ボトムシートに統合

2. **検索ボトムシート**（中央の検索ボタンタップで表示）
   - タイトル検索バー
   - 「または」区切り
   - 気分で探す（気分・時間・集中度選択）
   - 「作品を探す」ボタン
   - タイトル検索結果リスト + [+追加]ボタン

3. **結果画面**
   - 条件に合う作品リスト
   - 配信終了警告表示
   - 「作品を追加する」導線

4. **作品詳細画面**
   - ポスター
   - メタ情報（年、ジャンル、時間）
   - 配信終了警告
   - 配信サービス一覧（タップで各サービスへ）
   - 気分タグ（編集可能）
   - あらすじ
   - メモ欄（任意）
   - ステータス変更ボタン

5. **ステータス変更モーダル**
   - 見終わった → 評価・感想入力へ
   - 合わなくてやめた
   - 見たいに戻す
   - 削除

6. **追加モーダル**
   - 気分タグ選択（自動判定済み、変更可能）
   - 配信情報・終了予定表示
   - 追加ボタン

7. **マイリスト画面**
   - タブ切り替え（見たい/視聴中/見た）
   - 並び替え・絞り込み
   - 見たい: 終了まもなくを上部に
   - 視聴中: 配信サービス、メモ
   - 見た: 評価、離脱マーク

8. **フィード画面**
   - 友達のアクティビティ一覧
   - 見た/視聴中/見たい追加/おすすめ
   - 「見たいに追加」ボタン
   - コメント機能

9. **プロフィール画面（自分）**
   - アイコン、名前、ID
   - フォロー/フォロワー数
   - 統計（見た/視聴中/見たい数、今月・今年の視聴数）
   - 最近見た作品
   - 招待リンク生成

10. **プロフィール画面（他人）**
    - アイコン、名前、ID
    - フォロー/フォロワー数
    - フォローボタン
    - 統計
    - 公開設定に応じたリスト表示

11. **フォロー/フォロワー一覧**
    - タブ切り替え
    - ユーザーリスト
    - フォロー状態表示

12. **設定画面**
    - アカウント設定
    - プライバシー設定（リスト公開スコープ）
    - 通知設定
    - ログアウト/アカウント削除

13. **通知画面**
    - 日付グループ（今日/昨日/今週）
    - 配信終了アラート
    - フォロー通知
    - おすすめ通知
